# US-004 - Footer de navigation mobile

## Story
En tant qu'utilisateur mobile, je veux un footer fixe avec acces rapide aux sections principales.

## Criteres d'acceptation
1. Le footer est visible en bas de l'ecran sur les pages principales.
2. Le footer permet d'acceder a Accueil, Recherche, Chat, Bibliotheque et Profil.
3. Le footer reste visible en plein ecran et ne masque pas les messages d'erreur critiques.
4. Le style reste coherent desktop/mobile.

## Notes techniques
- Front: `apps/web/public/footer.js`, `apps/web/public/styles.css`
